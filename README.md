onsen.io
====

![Dependency Status](https://david-dm.org/OnsenUI/onsen.io.png)

The site for Onsen UI.

Installation
----

```
$ git clone https://github.com/OnsenUI/onsen.io.git
$ cd onsen.io
$ npm install gulp -g
$ npm install
$ git submodule init
$ git submodule update
```

Build
----

```
$ gulp generate --lang en
$ gulp generate --lang ja
```

Edit
----

```
$ gulp serve --lang en
```
