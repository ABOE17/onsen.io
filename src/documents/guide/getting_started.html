Getting Started

Onsen UIはモバイルHTML5アプリのユーザーインターフェイスを開発するためのUIフレームワークです。

Onsen UIはJSフレームワークのAngular.jsとCSSフレームワークのTopcoatをベースにして開発されており、Angular.jsのパワーとTopcoatのテーマををフルに活かすことができます。

また、Onsen UIは次のモバイルOSやブラウザでテストされています。

  • Android2.3.4+
  • iOS6+
  • Google Chrome
  • Safari

このページでは、初めてOnsen UIを利用する人のためにOnsen UIを導入して試す手順を説明します。

2. Run the project

Extract the zip file you just downloaded.

Open the terminal (command line) and cd to the extracted folder.

$ node scripts/web-server.js

Then navigate your browser to http://localhost:8000/app/index.html

Onsen UIはコマンドラインツールにNode.jsを使っています。Node.jsをインストールするには、nodejs.orgからパッケージをインストールして下さい。

3. Start adding your awesome app feature

Your project root is "app" folder. Use your favorite IDE and start adding awesome feature to your app.

You may want to download the Demos for sample codes.

To run the demo

Open the terminal (command line) and cd to the extracted folder.

$ node scripts/web-server.js

Then navigate your browser to http://localhost:8000/demo/index.html

4. Build your app

You can use Cordova to build Onsen UI project. Type in terminal: 

$ npm install -g cordova
$ cordova create MyApp && cd MyApp

copy all the files in the app folder of the template to MyApp/www folder

$ cordova platform add android
$ cordova build android
$ cordova emulate android
