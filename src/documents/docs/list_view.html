List View

モバイルアプリでは、上記のようなリストビューをよく用います。リストビューは、不特定多数の要素を上から下に並べて見せるのによく用いられます。このページでは、Onsen UIのons-list要素とons-list-item要素を使ってリストビューを扱う方法を記述します。

リストビューを表示する

ons-list要素は、ons-page要素や通常のページ下に置くことができます。ons-list要素の直下には必ずons-list-item要素を置きます。基本となるコードは以下です。

<!-- index.html -->
<ons-screen page="listview.html" > </ons-page>

<!-- listview.html -->
<ons-page>
  <ons-list>
    <ons-list-item>
      List Item
    </ons-list-item>
    <ons-list-item>
      List Item
    </ons-list-item>
  </ons-list>
</ons-page>

配列の内容を表示する

JavaScriptのデータをリストビューに表示するには、Angular.jsのコントローラを定義したのち、ons-list-item要素の属性にng-repeat属性を指定して以下のように記述します

// TODO
// app.js
var app = angular.module('app', ['angular-touch', 'onsen.directives']);

app.controller('ListViewController', function($scope) {
  $scope.items = [
    'Item A',
    'Item B'
  ];
});

<!-- listview.html -->
// TODO: まともにコードを書く
<ons-page>
  <div ng-controller="ListViewController">
    <ons-list>
      <ons-list-item ng-repeat="item as items">
      </ons-list-item>
    </ons-list>
  </div>
</ons-page>

ng-repeat属性、ng-init属性の詳細は、Angular.jsのドキュメントを参照して下さい。
