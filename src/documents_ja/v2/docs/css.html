CSSコンポーネントについて

Onsen UIにおけるCSSコンポーネントは、Onsen UIのカスタム要素のビジュアルデザインを規定します。 CSSコンポーネントは、その名の通りCSSで実装されているコンポーネントです。

CSSコンポーネントはあくまでCSSのみで実装されているので、CSSレベルで可能な振る舞いしか実装されていません。 例えば、CSSコンポーネントはプラットフォームに対して静的ですが、カスタム要素はプラットフォームを判別して自動的にiOSかもしくはMaterial Designのデザインを表示します。

開発者はCSSコンポーネントを直接使うのではなく、カスタム要素やフレームワークレベルのコンポーネントを使って下さい。 どのCSSコンポーネントがどのカスタム要素と対応しているかは、各CSSコンポーネントの対応する要素の項目を参照して下さい。
