<%- @markdown => %> ## Onsen UIとVue.jsでハイブリッドアプリやモバイルWebアプリを開発しよう Onsen UIをVue.js内で使いやすくするためのバインディングを作成しました。Onsen UIは **Web Components** に準拠した、ハイブリッドアプリ開発のためのライブラリです。 ## npmからダウンロード このバインディングは [NPMにて公開](https://www.npmjs.com/package/vue-onsenui)されています。利用するためには、Onsen UIライブラリも必要です。 アプリに組み込むには、下記のようにnpmコマンドを用いてプロジェクトに加えてください。 ```bash npm install --save onsenui vue-onsenui ``` ## 簡単に利用できる ライブラリはVue向けのプラグインとして配布されているため、下記のように簡単にアプリで使用できます。 ```javascript import Vue from 'vue'; import VueOnsen from 'vue-onsenui'; import ons from 'onsenui'; Vue.use(VueOnsen); ons.ready(() => { /* eslint-disable no-new */ new Vue({ el: 'body', components: { App } }); }); ``` また、いくつかのユーティリティ関数も含まれており、Onsen UIコンポーネントを簡単に扱うことができます。 ## 詳細を知るには ライブラリの使い方は、下記のドキュメント（英語）に記述しています。また、多くのOnsen UIコンポーネントを組み込んだサンプルアプリも提供しています。 * [Preview! Vue.js support for Onsen UI](/blog/preview-vue-support-onsen-ui/) * [Kitchensinkサンプル](https://github.com/OnsenUI/OnsenUI/tree/master/bindings/vue) * [GitHubレポジトリ](https://github.com/OnsenUI/OnsenUI/tree/master/bindings/vue) このライブラリはプレビュー段階のものですが、今後はドキュメントを増やたしたり、機能を追加していきます。Vue.jsを使ったモバイルアプリ開発を強力にサポートしていきます。 <% end %>
