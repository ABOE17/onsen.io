Onsen UI for Vue

iOS / Android のデザインを忠実に再現した、モバイル UI ライブラリの決定版。
はじめよう

iOS / Android ガイドラインに準拠した
34 個のコンポーネント

Onsen UI は iOS や Android のデザインを忠実に再現 したコンポーネントを提供します。

これにより、Web サイトや Progressive Web App (PWA)、および Cordova アプリを本物の iOS アプリ・Android アプリのような見た目に仕上げることができます。

複雑な UI の実装を驚くほど簡単に

Onsen UI は美麗なコンポーネントだけでなく、スタックナビゲーター、サイドメニュー、タブバー、カルーセル、無限スクロール、プルフック等といったモバイル特有の UI を簡単に実装するための仕組みも備えています。外部ページの動的な読み込みやキャッシュにも対応しています。

面倒な設定は不要

Onsen UI for Vue の利用に特別な知識は一切必要ありません。Vue.use(VueOnsen); と書くだけで、多種多様なコンポーネントをすぐに使えるようになります。

<%- @markdown => %> ```bash # Install two packages npm install onsenui vue-onsenui --save ``` ```js // Webpack CSS import import 'onsenui/css/onsenui.css'; import 'onsenui/css/onsen-css-components.css'; // JS import import Vue from 'vue'; import VueOnsen from 'vue-onsenui'; // This already imports 'onsenui' // And just plug the bindings Vue.use(VueOnsen); new Vue({ el: '#app', template: 'Click Me', }); ``` <% end %>

Vue CLI 用テンプレートを完備

Onsen UI for Vue をすぐに試したいという方のために、Vue CLI 用のテンプレートを2種類用意しています。 現在、Progressive Web App と Cordova アプリに対応しています。

また、外部サービスの Monaca CLI（無料）を利用すると Vue CLI で作成した Cordova アプリを実機でリアルタイムデバッグしたり、iOS / Android 向けにリモートでビルドしたりすることができます。

<%- @markdown => %> * __Progressive Web App__: [vue-pwa-webpack](https://github.com/OnsenUI/vue-pwa-webpack) ```bash $ vue init OnsenUI/vue-pwa-webpack hello-world ``` * __Cordova__: [vue-cordova-webpack](https://github.com/OnsenUI/vue-cordova-webpack) ```bash $ vue init OnsenUI/vue-cordova-webpack hello-world ``` <% end %>
