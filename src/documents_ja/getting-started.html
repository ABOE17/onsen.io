Onsen UIで美しいHTML5モバイルアプリを開発しよう!

Monaca CLIを使う MonacaクラウドIDEを使う NPM・Bowerパッケージ すべてダウンロード

ローカル環境でCordovaハイブリッドアプリをセットアップするには、Monaca CLIを利用すると便利です。Monacaを使うと、リモートビルドやデバッグ機能がクラウドを通じて提供されます。必要に応じてCordova CLIを使うこともできます。

  1. Monaca CLIをインストールする

    まずはNode.jsをインストールしてください。次に、Onsen UIアプリの開発のために、npmを通じてMonaca CLIをインストールします。GUI版デスクトップアプリは、WindowsとMac OS Xに対応したMonaca Localkitをご利用ください。

    npm install -g monaca
    セットアップで問題が発生する場合は、より手軽なMonacaクラウドIDEをご利用ください。

  2. テンプレートからプロジェクトを作成

    テンプレートからOnsen UIアプリを作成します。利用するフレームワークやOnsen UIのバージョンを選択できます。

    
              monaca create helloworld

  3. 実機上ですぐに確認

    ブラウザー上で確認するには、monaca previewを利用します。端末上にインストールしたデバッガー上で動かすには、monaca debugコマンドが便利です。

    
              cd helloworld
              monaca preview # ブラウザー上でプレビュー
              monaca debug # 実機デバッガー上で動作確認
              monaca remote build # アプリのビルド

MonacaクラウドIDEは、デバッグやリモートビルド機能を兼ね揃えたクラウドベースの開発環境です。

  1. Monacaにサインアップ

    Monacaアカウントを作成し、ダッシュボードにアクセスしてください。

    Monacaにサインアップ (無料)

  2. 新しいプロジェクトを作成

    新しいプロジェクトをクリックし、Onsen UIテンプレートを選択してください。

  3. Cloud IDEを開く

    ダッシュボードから開くをクリックすると、クラウドIDEが立ち上がり、開発をスタートできます。

  4. デバッガー実行やビルドを行う

    Monaca Debuggerを使うと、実機上で簡単に動作確認を行えます。

    リモートビルドを用いて、Android、iOS、Windows向けネイティブアプリ形式のパッケージを作成できます。

Onsen UIはnpmとBowerを通じて配信をしています。

  • NPMを用いたダウンロード

    下記のようにして、npmを用いてインストールできます。

    npm install onsenui

    Onsen UIはCommonJSとAMDの両方のフォーマットに対応しています。Browserify、RequireJSやWebpackを用いてアプリに組み込んでください。

    Reactエクステンションはreact-onsenuiパッケージとして提供されています。詳細についてはReactガイド（英語）を参照してください。

  • Bowerを用いたダウンロード

    bowerを用いて配布パッケージをインストールできます。

    bower install onsenui

  1. 配布パッケージ（リリースビルド）

    すべてのリリースビルドはOnsenUI Distribution Releasesにてダウンロードできます。

  2. ソースコード

    ソースコードの入手と、ビルドを行う方法については、GitHubレポジトリを参照してください。

    最新のmasterブランチをダウンロード
