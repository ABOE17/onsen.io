Onsen UI for Vue

Brings the Power and Simplicity of Vue.js to Hybrid and Progressive Web Apps.
Get Started

Beautifully designed UI components that ensure native look and feel

Onsen UI provides wide variety of ready-to-use UI components that emulates iOS and Android UI guidelines and designs. Building native looking high-performance apps could never be easier.

Easy to build even the most complicated UIs

In addition to pretty components, Onsen UI equips you with even powerful components like stack navigator, side menu, tab bar, carousel, infinite scroll and pull hook that makes it easy to implement mobile specific UIs.

No complicated setup

Just writing Vue.use(VueOnsen); will get you access to wide range of UI components.

<%- @markdown => %> ```bash # Install two packages npm install onsenui vue-onsenui --save ``` ```js // Webpack CSS import import 'onsenui/css/onsenui.css'; import 'onsenui/css/onsen-css-components.css'; // JS import import Vue from 'vue'; import VueOnsen from 'vue-onsenui'; // This already imports 'onsenui' // And just plug the bindings Vue.use(VueOnsen); new Vue({ el: '#app', template: 'Click Me', }); ``` <% end %>

CLI, Live reload, debugger, native plugins - all covered

You can get started with Onsen UI using our templates with Vue CLI.

Beyond that, we’ve got a full stack tools to streamline your app development with zero-time setup. Onsen UI family’s Monaca CLI and desktop app (for Windows and macOS) simplifies complicated tasks. Our development stack includes TypeScript, WebPack and Cordova, all integrated with debugging app with live-reload on device, remote-build and many other cloud services.
