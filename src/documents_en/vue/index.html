Onsen UI for Vue

Brings the Power and Simplicity of Vue.js to Hybrid and Progressive Web Apps.
Get Started

Beautifully designed UI components that ensure native look and feel

Onsen UI provides a wide variety of ready-to-use UI components that emulate iOS and Android UI guidelines and designs.

Simply pick out the elements you want in your app (like toolbars, lists, buttons... you name it) and they automatically show the iOS or Android look and feel according to the device they are running in.

Building native-looking and high-performance apps could never be easier.

Follow this link for a complete reference of all the components you can use in Onsen UI for Vue.

Easy to build even the most complicated UIs

In addition to pretty components, Onsen UI even equips you with powerful components like stack navigator, side menu, tab bar, carousel, infinite scroll and pull hook that make it easy to implement mobile specific UIs.

Now your app is beautiful AND easy to use.

Simple setup

No experience in using mobile UI frameworks? No problem. Getting started is not complicated, even for beginners:

  • (1) Just install our packages from the command line...
  • (2) ... import our files to your code...
  • (3) ... and write Vue.use(VueOnsen);

3 simple steps that will get you access to a wide range of UI components.

<%- @markdown => %> ```bash # (1) Install two packages $ npm install onsenui vue-onsenui --save ``` ```js // (2) Webpack CSS import import 'onsenui/css/onsenui.css'; import 'onsenui/css/onsen-css-components.css'; // (2) JS import import Vue from 'vue'; import VueOnsen from 'vue-onsenui'; // (3) And plug in the bindings Vue.use(VueOnsen); // Now you can use the awesome Onsen UI + Vue components new Vue({ el: '#app', template: 'Click Me', }); ``` <% end %>

You can get started with Onsen UI by using our templates within the Vue CLI.

Beyond that, we’ve got full-stack tools to streamline your app development with zero-time setup. Onsen UI family’s Monaca CLI and desktop app (for Windows and macOS) simplifies complicated tasks.
