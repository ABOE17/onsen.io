For instance, you can catch a tap event by using `onClick` event handler, or the text box change by `onChange` event handler. The following example describes how to get events that are native to React. ``` javascript class MyPage extends React.Component { handleClick() { ons.notification.alert('Hello, world!'); } render() { return() ( ); } } ``` ### Using DOM events Many of Onsen UI components will trigger additional DOM events. For instance, `Ons.Navigator` has `postpop` event which fires after a page is popped (become hidden). To catch those events, you can attach a DOM event like the example below. ``` var MyNavigator = React.createClass({ updateInformation: function(e) { // Do some work }, componentDidMount: function() { window.addEventListener('postpop', this.updateInformation); }, componentWillUnmount: function() { window.removeEventListener('postpop', this.updateInformation); }, // More code to satisfy Ons.Navigator Component }); ReactDOM.render(, document.getElementById('app')); ``` `componentDidMount` is called after the component is mounted, and has a DOM representation. Likewise, `componentWillUnmount` is called when the component is about to get unmounted. Event callback is bound to the react component and not the original element.
