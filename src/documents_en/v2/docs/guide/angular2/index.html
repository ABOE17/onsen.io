The document is based on Angular 2 RC + Onsen UI 2 RC + Onsen 2 for Angular 2 Beta. ### Starting Project Onsen UI for Angular 2 is delivered as NPM package ([`onsenui`](https://www.npmjs.com/package/onsenui) and [`angular2-onsenui`](https://www.npmjs.com/package/angular2-onsenui)). For easier installation, Monaca CLI will download all development & build dependencies including TypeScript compiler, Webpack and polyfills. ##### Use Monaca CLI ```bash $ npm install -g monaca # Install Monaca CLI - Onsen UI toolkit $ monaca create helloworld # Choose Angular 2 template $ cd helloworld; monaca preview # Run preview, or "monaca debug" to run on your device ``` ##### Download via npm ```bash # You need to install Onsen UI and Onsen UI for Angular 2 $ npm install onsenui angular2-onsenui ``` ### Loading Onsen UI in Angular 2 You need to import `onsenui` and `angular2-onsenui` package as shown below. ```javascript // Angular 2 imports import '@angular/platform-browser-dynamic'; import '@angular/platform-browser'; import '@angular/core'; import '@angular/http'; import '@angular/router'; // Onsen UI import 'onsenui' import {ONS_DIRECTIVES} from 'angular2-onsenui'; ``` ### Using Components Onsen UI components are Angular 2 directives, so just use them in the templates. ```html
