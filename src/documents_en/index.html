Onsen UI is an UI framework & tools to create native mobile apps with HTML5 hybrid app technology. As performant as native widgets, and much easier by using JavaScript and comprehensive development cloud.

The fastest way to develop beautiful native apps

Experience streamlined development with zero-time setup, with the skills you already have and in love with - JavaScript. With all the development stack provided on cloud, no more waste of time to maintain the platform SDKs which are always updating.

All tools you need in the Cloud

Powerful CLI (command-line tool) and desktop app (for Windows and macOS) simplifies complicated tasks. All integrated with debugging app with live-reload feature, remote-build and much more cloud services.

Beautiful native look and feel with lots of ready to use components with automatic styling

A rich variety of UI components specially designed for mobile apps. Rich components for tab, side menu and stack navigation. They all have iOS and Android Material design support, with automatic styling.

Performance optimized for mobile

A rich variety of Angular 2 components and directives specially designed for mobile apps. Rich components for tab, side menu and stack navigation. The components also expose a simple but powerful API complete with TypeScript definitions.

Easy to learn

We have community, tutorials, forum etc. All in the same common core.
