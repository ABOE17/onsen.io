    Getting started with Onsen UI for React

    Ask in our Forum Star on GitHub

    Start building amazing apps with Onsen UI and React

    Develop with Monaca CLI Use Monaca Cloud IDE Install via NPM or Bower Download Everything

    1. Install Onsen UI via Monaca CLI

      First install Node.js. Then, install Monaca CLI via npm, which provides command-line interface for Onsen UI development. Or, if you prefer GUI, download and install Monaca Localkit desktop app for Windows and OS X.

      npm install -g monaca Trouble setting up your environment? See Getting Started with Monaca Cloud IDE to start writing code right away!

    2. Create a project

      Create Onsen UI app from various templates and examples.

      monaca create helloworld --for="react"

    3. Run on your device

      monaca debug command will install the debugger on your device and run your app right away.

      cd helloworld
      monaca debug

    4. Visit Onsen UI Tutorial for React

      Visit our interactive tutorial site to learn more about Onsen UI and React extension!
