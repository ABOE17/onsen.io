Welcome to the new generation of UI Component Library.

Onsen UI 2.0 is almost here. It brings more components, native look & feel, delivered as Web Components, implemented in pure ES6. Also, it works well with AngularJS, React, jQuery and a bunch of other app frameworks.

We can't wait to see your awesome apps!

About Onsen UI 2.0

Onsen UI is an open source front-end framework for hybrid mobile apps and Websites with native look and feel. It is a part of Monaca Cloud platform, a collection of development tools, including a debugger and remote build service.

Onsen UI 2 will bring all the fancy features of Onsen UI 1, but we are making fundamental update to it. To make Web application more easier to develop, we thought we should not stick to a specific framework, but more to a framework agnostic attitude so that everyone can use our framework combined with their favorite application frameworks out there. Onsen UI 2 is future proof, framework agnostic, and designed to adapt the latest design trend.

Friendly with Angular 1 & 2, React and All Others

Onsen UI is fully independent; You can easily plug the Onsen UI components into any project, regardless of JavaScript framework. AngularJS users, we keep full support with all ng-* directives implemented as a binding.

Adapts Latest Design Trends for iOS and Android

Onsen UI 2.0 finally brings Android Material design in addition to the iOS flat design. Not only do the components have both styles, we also added many unique components to make your apps more materialized.

Responsive with New Splitter Component

New is a swiss-army knife for layouting your app. Your app can easily transition between sliding menu or a column-based view based on the user's device.

Future Proof and Compact with ES 6 and Web Components

All Onsen UI codes are re-written from scratch using the latest Web standard technologies. That enabled non-dependant framework implemented in pure JavaScript.

Getting Started

1. Quickstart

We have created a quickstart repository for you to get up and running instantly. The repository contains the latest beta version, with some Cordova based templates.

To get started, simply clone and serve the project.

$ git clone http://github.com/OnsenUI/onsenui2-quickstart
$ cd onsenui2-quickstart
$ npm install
$ gulp serve

2. Get Onsen UI 2.0 beta

With bower:

$ bower install onsen#2.0.0-beta

With npm:

$ npm install onsen@2.0.0-beta
Onsen UI 2.0 can be used with browserify: 
var onsen = require('onsenui/js/onsenui');

onsen.ready(function() {
  onsen.notification.alert({message: 'Onsen UI works with browserify!'});
});

3. Develop Your App

You can use Cordova CLI, PhoneGap or any other tools for Web or hybrid app development. However, Monaca makes you even easier to debug and build Onsen UI hybrid apps! Register your account on Monaca Cloud (free) and you can start developing right away.

Develop on browser using Monaca Cloud IDE:

With Monaca Cloud IDE, all debugging & remote build can be done online. To create a Onsen UI 2.0 project, please find "Onsen 2 Quick Start" template.

Develop on command line using Monaca CLI:

$ npm install -g monaca
$ monaca create myproject # And find template named "Onsen 2 Quick Start"

Documentation for Onsen 2.0 is still ongoing. Meanwhile, please take a look at our API reference or contact us if you have any questions!
